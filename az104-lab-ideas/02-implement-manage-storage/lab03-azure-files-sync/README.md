# Lab 03: Azure Files Sync

## Overview
This lab focuses on configuring Azure File Sync to synchronize on-premises file servers with Azure Files.

## Learning Objectives
- Create Azure File Shares
- Install and configure Azure File Sync agent
- Set up sync groups and cloud endpoints
- Monitor sync health and troubleshoot issues

## Prerequisites
- Azure subscription with appropriate permissions
- Windows Server for testing (or Azure VM)
- Understanding of file server concepts
