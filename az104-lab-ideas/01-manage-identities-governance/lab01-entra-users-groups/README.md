# Lab 01: Entra Users and Groups

## Overview
This lab focuses on managing users and groups in Microsoft Entra ID (formerly Azure Active Directory).

## Learning Objectives
- Create and manage users in Entra ID
- Create and manage groups
- Assign users to groups
- Understand group types (Security vs. Microsoft 365)

## Prerequisites
- Azure subscription with appropriate permissions
- Access to Microsoft Entra ID
