# Azure 104 Lab Ideas

This repository contains hands-on lab exercises for people learning Azure 104 certification topics.

## Lab Structure

### 01 - Manage Identities and Governance
- **Lab 01**: Entra Users and Groups
- **Lab 02**: RBAC Custom Roles
- **Lab 03**: Subscriptions and Management Groups

### 02 - Implement and Manage Storage
- **Lab 01**: Storage Accounts
- **Lab 02**: Blob Lifecycle Management
- **Lab 03**: Azure Files Sync

## How to Use These Labs

Each lab directory contains exercises and resources to help you practice Azure 104 concepts. Navigate to the specific lab folder to get started.
