# Lab 02: RBAC Custom Roles

## Overview
This lab focuses on creating and managing custom Role-Based Access Control (RBAC) roles in Azure.

## Learning Objectives
- Understand Azure RBAC concepts
- Create custom RBAC roles
- Assign roles to users and groups
- Test and validate custom role permissions

## Prerequisites
- Azure subscription with appropriate permissions
- Understanding of Azure RBAC fundamentals
