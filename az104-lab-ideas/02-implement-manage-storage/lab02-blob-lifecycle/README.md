# Lab 02: Blob Lifecycle Management

## Overview
This lab focuses on implementing lifecycle management policies for Azure Blob Storage.

## Learning Objectives
- Understand blob storage tiers (Hot, Cool, Archive)
- Create lifecycle management policies
- Configure automatic tier transitions
- Implement blob deletion rules

## Prerequisites
- Azure subscription with appropriate permissions
- Existing storage account with blob containers
