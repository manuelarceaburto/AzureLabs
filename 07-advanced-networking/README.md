# Module 07: Advanced Networking

## Overview

This module covers advanced Azure networking services including Azure Firewall, VPN Gateway, and Application Gateway. These labs build upon the foundational networking concepts from Module 04.

## Prerequisites

- Completed Module 04: Configure and Manage Networking
- Active Azure subscription with Contributor access
- VS Code with Azure extensions installed (Module 00)
- Basic understanding of VNets, subnets, and NSGs

## Labs in This Module

### Lab 01: Azure Firewall and Firewall Manager
**Duration:** 60 minutes  
**Objectives:**
- Deploy Azure Firewall
- Configure application and network rules
- Implement forced tunneling
- Monitor firewall logs

### Lab 02: VPN Gateway and Site-to-Site Connectivity
**Duration:** 75 minutes  
**Objectives:**
- Create VPN Gateway
- Configure site-to-site VPN
- Implement point-to-site VPN
- Troubleshoot VPN connectivity

### Lab 03: Application Gateway with Web Application Firewall
**Duration:** 60 minutes  
**Objectives:**
- Deploy Application Gateway
- Configure backend pools and routing rules
- Enable Web Application Firewall (WAF)
- Implement SSL termination

## Learning Path

1. **Start with Lab 01** - Understand network security with Azure Firewall
2. **Progress to Lab 02** - Learn hybrid connectivity patterns
3. **Complete with Lab 03** - Master application-level load balancing and security

## Module Completion

After completing all labs, you will be able to:
- ✅ Design and implement Azure Firewall solutions
- ✅ Configure site-to-site and point-to-site VPN connections
- ✅ Deploy and manage Application Gateway with WAF
- ✅ Troubleshoot advanced networking scenarios
